toolbox::utils::SimpleTimerArray class
#include <timer.hpp>

Concrete implementation of SimpleTimerInterface using an array of timers.

Contents

  • Reference
    • Base classes
    • Constructors, destructors, conversion operators
    • Public functions

This class implements the SimpleTimerInterface using a vector of StopWatchTimer objects. It provides thread-safe operations on multiple timers.
