Splits a string view into a vector of strings based on a string delimiter.

Splits a string view into a vector of strings based on a string delimiter.Splits the input string at each occurrence of the delimiter string. Empty parts between delimiters are included in the result. The delimiter itself is not included in the output.

// Split by string delimiter auto result = split("apple,orange,,banana", ","); // result will be {"apple", "orange", "", "banana"}
