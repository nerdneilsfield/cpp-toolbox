template<typename Iterator, typename Function>
CPP_TOOLBOX_EXPORT void toolbox::concurrent::parallel_for_each(Iterator begin, Iterator end, Function func)

并行地对范围 [begin, end) 中的每个元素应用函数 func。

Template parameters
Iterator 输入迭代器类型（需要支持随机访问以进行高效划分）。
Function 一元函数对象类型。
Parameters
begin 范围的起始迭代器。
end 范围的结束迭代器。
func 应用于每个元素的函数对象。

使用默认的线程池进行手动任务划分和提交。

template<typename InputIt, typename OutputIt, typename UnaryOperation>
CPP_TOOLBOX_EXPORT void toolbox::concurrent::parallel_transform(InputIt first1, InputIt last1, OutputIt d_first, UnaryOperation unary_op)

并行地对输入范围 [first1, last1) 应用 unary_op，并将结果存储到 [d_first, ...)。

Template parameters
InputIt 输入迭代器类型 (std::random_access_iterator)。
OutputIt 输出迭代器类型 (std::random_access_iterator)。
UnaryOperation 一元操作函数对象类型, 签名应类似 OutType(const InType&)。
Parameters
first1 输入范围的起始。
last1 输入范围的结束。
d_first 输出范围的起始。
unary_op 应用于每个输入元素的操作。

Warning

要求输入和输出迭代器都必须是【随机访问迭代器】。

输出范围 [d_first, d_first + distance(first1, last1)) 必须具有足够的、已分配的容量，并且并行写入必须是安全的（例如，写入 std::vector 前需 resize()）。

template<typename Iterator, typename T, typename BinaryOperation>
CPP_TOOLBOX_EXPORT T toolbox::concurrent::parallel_reduce(Iterator begin, Iterator end, T identity, BinaryOperation reduce_op)

并行地对范围 [begin, end) 进行归约操作。

Template parameters
Iterator 输入迭代器类型（需要支持随机访问以进行高效划分）。
T 归约结果和幺元的类型。
BinaryOperation 归约操作的二元函数对象类型，签名应类似 T(const T&, const ElementType&) 或 T(T, T)。
Parameters
begin 范围的起始迭代器。
end 范围的结束迭代器。
identity 归约操作的幺元。
reduce_op 用于合并两个 T 类型值或 T 类型值与元素类型值的二元操作。
Returns 并行归约的结果。

使用默认线程池进行手动任务划分，先进行局部归约，然后合并结果。 reduce_op 最好满足结合律 (associative)。identity 必须是 reduce_op 的幺元。
