Detailed Description

template<typename DataType, typename KNNSearcher = kdtree_t<DataType>>
class toolbox::pcl::point_to_plane_icp_t< DataType, KNNSearcher >

Point-to-Plane ICP 算法实现 / Point-to-Plane ICP algorithm implementation.

该算法最小化源点到目标点切平面的距离。需要目标点云具有法线信息。 This algorithm minimizes the distance from source points to target point tangent planes. Requires target point cloud to have normal information.
